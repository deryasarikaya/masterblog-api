# Masterblog API

A RESTful API built with Flask for managing blog posts.

## Features
- List all posts (GET)
- Add a new post (POST)
- Update a post (PUT)
- Delete a post (DELETE)
- Search posts (GET)
- Sort posts (GET)
- API Documentation with Swagger

## Installation

1. Clone the repository:
git clone https://github.com/deryasarikaya/masterblog-api.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the backend:
python3 backend/backend_app.py

4. Run the frontend:
python3 frontend/frontend_app.py

## API Documentation
Open http://localhost:5002/api/docs in your browser.
